# BitK

Download [bitcoin core](https://github.com/bitcoin-s/bitcoin-s) and create a config for testnet as below

Bitcoin config

- testnet=1
- prune=1000
- server=1
- test.rpcport=16666
- test.rcp.port=16669
- rpcuser=user123
- rpcpassword=password123
